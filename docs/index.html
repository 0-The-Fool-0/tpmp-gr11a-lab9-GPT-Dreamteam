Документация BankApp

Разработка мобильного приложения для банка, предоставляющего клиентам следующие возможности:

  • Аутентификация по логину/паролю
  • Просмотр всех активных и заблокированных банковских счетов (закрытые счета не отображаются)
  • Просмотр курсов валют
  • Поиск ближайших отделений банка на карте
  • Локализация на русский язык

Функциональные требования

IDТребование
FR-01Приложение должно обеспечивать аутентификацию пользователя по логину и паролю
FR-02После успешного входа пользователь переходит в главное меню
FR-03Пользователь может просмотреть список всех своих счетов (активных и заблокированных)
FR-04Закрытые счета не отображаются в списке
FR-05Система поддерживает типы счетов: текущий, сберегательный, кредитный, карт-счет
FR-06Карт-счет может быть: зарплатным, сберегательным, кредитным
FR-07Для зарплатных карт отображается информация об овердрафте
FR-08Пользователь может просматривать текущие курсы валют
FR-09Пользователь может просматривать карту с филиалами отделений
FR-10Пользователь может найти ближайшее отделение относительно своего местоположения
FR-11Все данные хранятся в SQLite базе данных
FR-12При первом запуске база данных заполняется тестовыми данными

Нефункциональные требования

  • Производительность: время отклика UI не более 300 мс
  • Надёжность: корректная обработка ошибок сети и геолокации
  • Платформа: iOS 16.0+
  • Архитектура: MVVM + Coordinator (Clean Swift)
  • Локализация: русский язык
